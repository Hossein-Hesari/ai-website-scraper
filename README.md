# 🤖 AI-Powered Website Scraper

[![Python](https://img.shields.io/badge/Python-3.x-blue.svg)](https://www.python.org/)
[![Selenium](https://img.shields.io/badge/Selenium-Automation-green.svg)](https://www.selenium.dev/)
[![OpenAI](https://img.shields.io/badge/OpenAI-API-orange.svg)](https://openai.com/)

این پروژه یک اسکرِپر هوشمند است که با ترکیب قدرت **Selenium** (برای رندرینگ صفحات) و **OpenAI** (برای تحلیل محتوا)، چالش استخراج داده از سایت‌های مدرن و داینامیک را حل می‌کند. به جای نوشتن سلکتورهای دستی (XPath/CSS Selector) که با کوچکترین تغییر در UI سایت از کار می‌افتند، این ابزار تگ‌های مورد نظر شما را با هوش مصنوعی شناسایی می‌کند.

---

## 🚀 چرا این ابزار؟
بسیاری از سایت‌های امروزی از جاوا اسکریپت برای بارگذاری محتوا استفاده می‌کنند. اسکرپرهای سنتی که بر پایه `requests` هستند، در این سایت‌ها ناتوان‌اند. این پروژه:
*   **پشتیبانی از صفحات پویا:** با استفاده از Selenium، کدهای JavaScript را اجرا می‌کند تا صفحه به‌طور کامل رندر شود.
*   **هوش مصنوعی هوشمند:** به جای حدس زدن سلکتورها، HTML صفحه را به AI می‌دهد تا دقیقاً همان چیزی که می‌خواهید را پیدا کند.
*   **انعطاف‌پذیری بالا:** قابلیت کلیک روی دکمه‌ها، لینک‌ها و حتی وارد کردن متن در فیلدهای `input`.

---

## 🛠 ابزارهای مورد استفاده
*   **Python:** زبان اصلی پروژه.
*   **Selenium:** جهت شبیه‌سازی مرورگر و رندرینگ صفحات داینامیک.
*   **OpenAI API:** جهت تحلیل ساختار HTML و شناسایی سلکتورهای پویا.

---

## 📋 راهنمای استفاده

### پیش‌نیازها
1. نصب درایورهای مرورگر (مانند ChromeDriver).
2. نصب کتابخانه‌های مورد نیاز (Selenium, OpenAI).
3. داشتن API Key از OpenAI (یا استفاده از WorkspaceID برای محیط‌های سازمانی).

### مراحل اجرا
1. **آماده‌سازی:** نوت‌بوک `notebook.ipynb` را در محیط دلخواه (مانند Jupyter یا VS Code) باز کنید.
2. **پیکربندی:** کلید API یا WorkspaceID خود را در بخش تنظیمات کد قرار دهید.
3. **اجرا:** سلول‌های مربوط به `scrape_with_ai` را اجرا کنید تا عملیات دریافت HTML و تحلیل توسط هوش مصنوعی انجام شود.

**نکات کاربردی:**
*   **تعامل با عناصر:** اگر عنصر شناسایی شده `button` یا تگ `a` باشد، هوش مصنوعی به‌صورت خودکار عملیات `click` را انجام می‌دهد.
*   **ورودی متن:** برای فیلدهای `input`، کافیست متغیر `text_to_type` را در پارامتر تابع `scrape_with_ai` مقداردهی کنید.
